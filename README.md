# Hello India Static Application

A simple static HTML page that displays a "Hello India" message, as per the requirement.

## Overview

This project contains a single `index.html` file that can be served by any web server or opened directly in a browser. It uses inline CSS for styling and has no external dependencies.

## How to Use

- **Directly in browser**: Open `index.html` in a web browser.
- **Via a web server**: Place the file in a directory served by Apache, Nginx, or any HTTP server.

## Technology Stack

- **HTML5**
- **CSS3** (inline)

## Design Decisions

- No JavaScript required (static content).
- No build tools or package managers.
- Minimalist and mobile-friendly.
